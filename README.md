# mupot
The MuPot is an add-on for the BBC microbit featuring: 
- two potentiometers hooked to analog inputs of the microbit, 
- an audio jack with volume potentiometer, 
- a coin-battery holder with on/off switch fro the microbit
- a big buzzer with a by-pass switch.

This repo includes:
1) gerber files for the PCB manufacturing
2) Three Mu-python scripts specific to the Mupot add-on.
-  a music synthetiser
-  a 25 pixels tetris game (work in progress).
-  a wifi messaging system with synthetic voice
