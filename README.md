# mupot

## Open-source hardware!
The MuPot is an open-source add-on for the BBC microbit featuring: 
- two potentiometers hooked to analog inputs of the microbit, 
- an audio jack with volume potentiometer, 
- a coin-battery holder with on/off switch for the microbit
- a big buzzer with a by-pass switch.
